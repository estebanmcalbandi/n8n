# Mi primer n8n

Sencillo contenedor con N8N.

## Características

- **Función principal:** Probar N8N usando un sencillo contenedor.
- **Tecnologías:** N8N.
- **Estado:** Estable.

## Requisitos
- **Sistema operativo:** Probado en Ubuntu.
- **Docker:** Necesita tener instalado previamente docker.
